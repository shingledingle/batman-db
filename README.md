# batman-db

Na na na na na na na na Batman

A PostgreSQL + Flask database for exploring the DC Universe and its illustrious characters, story arcs, and more.

## Features (Planned)
- Browse DC characters with stats, aliases, and affiliations
- Search story arcs and their character appearances
- Explore villain/hero relationships
- REST API for querying the database

## Tech Stack
- **Backend**: Python, Flask
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy (or raw SQL)
- **Frontend**: HTML/CSS (Jinja2 templates) or React
